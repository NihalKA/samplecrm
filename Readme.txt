How to run the Small CRM  Project
1. Download the  zip file

2. Extract the file and copy crm folder

3.configure lamp server in aws ubuntu server

4. Use mysql command line inside ubuntu server

5. Create a database with name crm

6. Import crm.sql file(given inside the zip package in SQL file folder)

7.Run the script http://ip-address/crm (frontend)

8.Run the script http://ip-address/crm/admin
